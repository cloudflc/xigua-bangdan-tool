# 榜单制作工具使用说明

## 功能介绍

本工具用于自动生成获奖榜单网页，支持以下功能：
- 读取名单.csv文件，自动生成完成奖、优秀奖、创意奖的获奖名单网页
- 可选读取小书童Excel文件，自动合并完成奖名单
- 选择输出路径，自动生成HTML网页文件

## 使用方法

1. **准备文件**
   - 准备一个`名单.csv`文件，格式如下：
     ```
     奖项	姓名
     完成奖	张三
     完成奖	李四
     优秀奖	王五
     优秀奖	赵六
     创意奖	小明
     ```
   - （可选）准备小书童Excel文件，包含学员名单

2. **运行工具**
   - 双击运行`榜单制作工具.exe`
   - 点击"浏览"按钮选择`名单.csv`文件
   - （可选）点击"浏览"按钮选择小书童Excel文件
   - 点击"浏览"按钮选择输出路径（生成的HTML文件将保存到该路径）
   - 点击"制作榜单"按钮

3. **查看结果**
   - 工具会自动生成以下HTML文件：
     - `完成奖.html` - 完成奖获奖名单
     - `优秀奖-创意奖.html` - 优秀奖和创意奖获奖名单（如果有创意奖）
     - `优秀奖.html` - 优秀奖获奖名单（如果没有创意奖）

## 注意事项

- 名单.csv文件必须使用制表符分隔（Tab分隔）
- 小书童文件是可选的，如果不提供，则只使用名单.csv中的完成奖
- 输出路径必须具有写入权限
- 生成的HTML文件可以直接在浏览器中打开查看

## 故障排除

如果遇到问题，请检查：
1. 名单.csv文件格式是否正确
2. 是否有足够的磁盘空间
3. 是否有输出路径的写入权限
4. 是否安装了必要的依赖项（如果从源代码运行）
